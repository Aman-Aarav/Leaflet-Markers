Leaflet Marker Examples

This repository contains multiple Leaflet.js marker examples, demonstrating both static markers and interactive click-based custom markers. These examples are useful for learning Leaflet basics, location selection, and map-based UI interactions.

Examples Included:

1. Simple Marker Example

        File: Leaflet-marker.html
        Displays a Leaflet map centered on Bengaluru, India
        Adds a single static marker
        Shows a popup when the marker is clicked
        Popup opens automatically on load

2. Custom Marker on Click Example

        File: Leaflet-custom-marker.html
        Click anywhere on the map to place a marker
        Uses a custom icon
        Automatically removes the previous marker
        Displays latitude & longitude in popup
        Full-screen responsive map
